Does it matter to use relative size(%,em,rem) for styling the size of an element so that the element is responsive ?


In my opinion, using relative size for styling the size of elements matter in our design. Em and rem are flexible, scalable units which are translated by the browser into pixel values, depending on the font size of your design. If a value of 1em or 1rem is used , the browser translates it to 16px - 160px or any other value. Using em and rem units gives us flexibility and the ability to scale elements up and down. The flexibility makes our design to become more responsive and to allow browser users to control the overall scale of sites for maximum readability. %, em, rem is useful because it is based off the default font which is not in fixed size.

The specific changes in CSS3


1. Animations and Transitions-there are two ways to create CSS animations.The first is to use CSS properties with the ‘transition’ declaration.The second way involves the description of specific moments of the animation with the ‘code>@keyframe ’ rule.
2. Calculating values with calc()-the calc() function allows you to do simple arithmetic calculations in CSS.
3. Advanced selectors-you don't need to specify classes and IDs for every element. Instead, you can utilize the attribute field to style them.
4. CSS 3 Gradient-help to create smooth transitions between colors without having to resort to images. They can be linear or radial, and can be set to repeat.
5. Box Images- ‘border-image’ property allows you to display custom designed borders around elements.
6. Media Queries- help to build sites that are responsive and which adapt to the type of device, orientation and resolution.
7. CSS Columns-column-based layouts are implemented by using the CSS ‘columns’ rule.
8. CSS 3 RGBA- CSS 3 introduces a modification and added opacity manipulation to this property. RGB has been transformed to RGBA (Red Green Blue Alpha channels), which simplifies how you control the opacity of elements.
9. CSS 3 Web Fonts - CSS 3 also facilitates embedding any custom font on a webpage.
10. CSS 3 Text Shadow-The new text-shadow property allows you to add shadows to the text on a webpage.
